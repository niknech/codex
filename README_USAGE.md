# README 2: Инструкция по использованию приложения

Этот файл описывает работу уже запущенного приложения `Telegram Keyword Cleaner`.

## 1) Авторизация

В верхней части окна заполните:
- **API ID**
- **API Hash**
- **Phone** (например, `+7999...`)

Далее:
1. Нажмите **«Авторизоваться»**.
2. Когда Telegram отправит код, введите его в поле **Code**.
3. Если включен облачный пароль, введите его в поле **2FA Password**.

После успешной авторизации приложение загрузит список чатов.

---

## 2) Выбор чатов для обработки

Доступны варианты:
1. **Конкретный чат** — выберите в выпадающем списке.
2. **Любой чат** — поставьте галочку **«Любой чат»** (обрабатываются чаты из основного списка).
3. **По username вручную** — введите username без `@`.

Если указаны одновременно username и выбран чат — приоритет у username.

---

## 3) Фильтр по датам

- **Дата от** — нижняя граница.
- **Дата до** — верхняя граница.

Можно оставить пустыми для сканирования всей доступной истории.

---

## 4) Режимы работы

- **Dry-run (не удалять)**
  - Ничего не удаляется.
  - Совпадения показываются в логах и записываются в `deleted_log.csv`.

- **Delete for all**
  - При реальном удалении приложение пытается удалить сообщения для всех (`revoke`).
  - Если Telegram/TDLib не разрешит такое удаление, сообщение будет пропущено и ошибка попадет в лог.

- **Показывать логи**
  - Включает/выключает вывод детальных логов в окне.

---

## 5) Запуск сканирования/удаления

1. Убедитесь, что `keywords.txt` заполнен.
2. Нажмите **«Сканировать и удалить»**.
3. Приложение постранично читает историю выбранных чатов.
4. Проверяется:
   - текст сообщений,
   - подписи (caption) у медиа.
5. Совпадение ищется по подстроке, регистр игнорируется, `ё` приравнивается к `е`.

---

## 6) Что записывается в CSV

В файл `deleted_log.csv` пишется каждая найденная/обработанная запись:

- `chat_id`
- `chat_title`
- `message_id`
- `date`
- `matched_keyword`
- `text_snippet`

`text_snippet` — укороченный фрагмент текста сообщения.

---

## 7) Рекомендованный безопасный сценарий

1. Сначала запустите с включенным **Dry-run**.
2. Проверьте результаты в UI-логах и `deleted_log.csv`.
3. Если всё верно — отключите Dry-run и повторите.
4. При необходимости включите **Delete for all**.

---

## 8) Ограничения и примечания

- Удаление зависит от прав в чате и правил Telegram.
- Не все сообщения можно удалить «для всех».
- Используйте инструмент только там, где у вас есть право на модерацию/удаление.
