# I8080: перевод 16-битного HEX числа в DEC

Небольшой учебный проект на ассемблере Intel 8080 (под CP/M), который:

1. Берёт 16-битное число из `HEX_VALUE`.
2. Преобразует его в десятичную строку из 5 цифр.
3. Выводит строку через BDOS (`CALL 5`, функция `9`).

## Файл проекта

- `hex16_to_dec.asm` — исходник программы.

## Как это работает

Преобразование выполняется без аппаратного деления:

- последовательно используются делители `10000`, `1000`, `100`, `10`, `1`;
- для каждого делителя число в `HL` многократно уменьшается на делитель;
- количество вычитаний даёт очередную десятичную цифру;
- остаток передаётся на следующий шаг.

Таким образом формируется строка вида `"43981"` для примера `0ABCDh`.

## Сборка (пример)

Используйте любой совместимый 8080-ассемблер (например, в составе эмулятора CP/M).

Примерный сценарий:

1. Ассемблировать `hex16_to_dec.asm`.
2. Запустить полученный COM-файл в CP/M.
3. На экране будет выведено десятичное значение.

## Изменение входного значения

В исходнике поменяйте константу:

```asm
HEX_VALUE:
    DW 0ABCDh
```

на нужное 16-битное значение.
