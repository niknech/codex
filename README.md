# Красивый live-дашборд из Excel

## Что делает проект
- Читает `data.xlsx` (должен лежать рядом с `app.py`).
- Каждые **1.5 секунды** добавляет по одной строке в дашборд.
- Обновляет:
  - один основной график **Расход измеренный** и **Расход предсказанный** от `Date` с единицей измерения `м³/ч` на оси Y;
  - таблицу **Средние значения** (под графиком): Абсолютная погрешность (`м³/ч`), Относительная погрешность (%), Расход по ИЛ (`м³/ч`).
- Если средняя относительная погрешность (`mean_mape`) становится выше **3%**, включается стильная цветовая сигнализация.

## Как запустить (очень просто)
1. Установи Python 3.10+.
2. В папке проекта выполни:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. Положи свой Excel-файл в корень проекта и назови его `data.xlsx`.
4. Запусти:
   ```bash
   python app.py
   ```
5. Открой в браузере: http://127.0.0.1:5000

## Как обновить код после этой доработки
Полностью весь алгоритм заново делать **не нужно**.

Если у тебя уже есть папка проекта и virtualenv:
```bash
git pull
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Если ты запускаешь с нуля на новом компьютере — просто выполни шаги из раздела «Как запустить».

## Важный формат колонок в Excel
`Date, Q_IL, Q_IL_pred, mape, mae, mean_mae, mean_mape, mean_Q_IL_pred`
