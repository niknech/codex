# I8080-подпрограмма HEX16 -> DEC (исполнение на Z80)

Подпрограмма рассчитана на выполнение на **Z80**, но написана в **мнемонике Intel 8080**.

## Интерфейс

- Вход: `HL` = 16-битное число (`0000H..FFFFH`).
- Вызов: `CALL START`.
- Выход: в `DEC_BUFFER` формируется десятичная строка с `00H` в конце.

Пример использования в основной программе:

```asm
        lxi h,1234h
        call START
        lxi h,DEC_BUFFER
        call puts
```

Для `1234H` результат в буфере: `"4660",00H`.

## Алгоритм

- Последовательно обрабатываются делители: `10000`, `1000`, `100`, `10`, `1`.
- Каждая цифра получается количеством вычитаний делителя из текущего остатка.
- Ведущие нули подавляются, но для `0000H` корректно выводится `0`.

## Данные

- `DEC_BUFFER` — буфер результата (до 5 цифр + `00H`).
- `OUT_PTR`, `STARTED`, `WORK_VALUE` — служебные переменные.
