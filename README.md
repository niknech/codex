# I8080-проект: HEX16 -> DEC (исполнение на Z80)

Программа рассчитана на выполнение на **Z80**, но написана в **мнемонике Intel 8080** (без Z80-специфичных инструкций).

## Что делает программа

1. Берёт 16-битное значение из `HEX_VALUE`.
2. Преобразует его в десятичную ASCII-строку.
3. Выводит результат через CP/M BDOS (`CALL 5`, функция `9`).

Файл исходника: `hex16_to_dec.asm`.

## Особенности реализации

- Используются только инструкции, совместимые с 8080.
- Алгоритм без аппаратного деления: последовательные вычитания делителей `10000`, `1000`, `100`, `10`, `1`.
- Ведущие нули подавляются (например, `0001H` печатается как `1`).
- Для нулевого значения корректно печатается `0`.

## Как собрать и запустить

1. Ассемблируйте `hex16_to_dec.asm` 8080-совместимым ассемблером.
2. Запустите полученный `.COM` в CP/M на Z80.

## Как задать своё число

Измените константу:

```asm
HEX_VALUE:
    DW 0ABCDH
```

на нужное 16-битное значение (`0000H..0FFFFH`).
