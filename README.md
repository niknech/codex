# Telegram Keyword Cleaner

Этот репозиторий содержит WPF-приложение для удаления сообщений в Telegram по ключевым словам через TDLib.

Документация разделена на два отдельных файла:

1. **Сборка и запуск (VSCode, для обычного пользователя)**: [README_SETUP.md](README_SETUP.md)
2. **Инструкция по использованию приложения**: [README_USAGE.md](README_USAGE.md)
