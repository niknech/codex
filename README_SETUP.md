# README 1: Сборка и запуск (для обычного пользователя с VSCode)

Ниже — пошаговая инструкция, если у вас есть только **VSCode** и вы хотите собрать/запустить приложение на Windows.

## 1) Что нужно установить

1. **Windows 10/11 x64**
2. **.NET 8 SDK**
   - Скачать: https://dotnet.microsoft.com/download/dotnet/8.0
3. **VSCode**
   - Скачать: https://code.visualstudio.com/
4. Расширение C# для VSCode (рекомендуется официальный пакет C# от Microsoft).

---

## 2) Скачайте проект

Варианты:
- Скачать ZIP архива репозитория и распаковать.
- Или через git:

```bash
git clone <URL_репозитория>
cd <папка_проекта>
```

---

## 3) Получите `api_id` и `api_hash`

1. Откройте https://my.telegram.org
2. Войдите по номеру телефона.
3. Перейдите в **API development tools**.
4. Создайте приложение и сохраните:
   - `api_id`
   - `api_hash`

---

## 4) Настройте `config.json`

Создайте/отредактируйте файл `config.json` в корне проекта:

```json
{
  "ApiId": 123456,
  "ApiHash": "ваш_api_hash",
  "PhoneNumber": "+79990000000"
}
```

---

## 5) Подготовьте `keywords.txt`

Создайте/отредактируйте `keywords.txt` в корне проекта (UTF-8, по одному ключевому слову на строку):

```text
спам
реклама
```

---

## 6) Подключите TDLib (важно)

NuGet-пакет `TdLib` уже указан в проекте, но для работы нужен нативный файл **`tdjson.dll`**.

### Где взять `tdjson.dll`
- Официальный репозиторий TDLib: https://github.com/tdlib/td
- Либо готовая Windows-сборка из доверенного источника.

### Куда положить
Положите `tdjson.dll`:
- либо рядом с собранным `.exe` в `bin/...`,
- либо в папку, которая входит в `PATH`.

Если `tdjson.dll` не найден, приложение не сможет стартовать корректно.

---

## 7) Сборка и запуск через VSCode

Откройте папку проекта в VSCode, затем откройте терминал (Terminal → New Terminal) и выполните:

```bash
dotnet restore
dotnet build -c Release
dotnet run -c Release
```

После запуска откроется окно WPF-приложения.

---

## 8) Где хранятся файлы приложения

- `tdlib_data/` — сессия и база TDLib
- `config.json` — API и номер телефона
- `keywords.txt` — ключевые слова
- `deleted_log.csv` — лог совпадений/удалений

---

## 9) Если не запускается

1. Проверьте, что установлен именно **.NET 8 SDK**:
   ```bash
   dotnet --version
   ```
2. Проверьте наличие `tdjson.dll` рядом с exe.
3. Убедитесь, что `ApiId/ApiHash` указаны корректно.
4. Проверьте формат `keywords.txt` (UTF-8).
